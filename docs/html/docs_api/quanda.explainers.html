class quanda.explainers.RandomExplainer(model: Module, train_dataset: Dataset, seed: int = 27, **kwargs)[source]

Bases: Explainer

The most basic version of a random explainer. The explanations are generated with independent values sampled from a uniform distribution in [0,1].

Methods
