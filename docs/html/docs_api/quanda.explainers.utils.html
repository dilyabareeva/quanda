quanda.explainers.utils module

quanda.explainers.utils.explain_fn_from_explainer(explainer_cls: type, model: Module, test_tensor: Tensor, train_dataset: Dataset, targets: List[int] | Tensor | None = None, **kwargs: Any) → Tensor[source]

Compute influence scores for the test samples using the specified explainer class.

Parameters:

  • explainer_cls (type) – The explainer class to use for computing explanations.

  • model (torch.nn.Module) – The model to be used for the influence computation.

  • test_tensor (torch.Tensor) – The test samples for which influence scores are computed.

  • train_dataset (torch.utils.data.Dataset) – Training dataset to be used for the influence computation.

  • targets (Optional[Union[List[int], torch.Tensor]], optional) – Labels for the test samples. Defaults to None.

  • cache_dir (str, optional) – Directory for caching results. Defaults to “./cache”.

  • model_id (Optional[str], optional) – Identifier for the model. Defaults to None.

  • **kwargs (dict) – Additional keyword arguments passed to the explainer.

Returns:

2D Tensor of shape (test_samples, train_dataset_size) containing the influence scores.

Return type:

torch.Tensor

  • **kwargs (dict) – Additional keyword arguments passed to the explainer.

torch.Tensor
