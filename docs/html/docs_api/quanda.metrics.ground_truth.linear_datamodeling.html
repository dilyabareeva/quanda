Metric to evaluate data attribution methods using the Linear Datamodeling Score (LDS).

The LDS measures how well a data attribution method can predict the effect of retraining a model on different subsets of the training data. It computes the correlation between the model‚Äôs output when retrained on subsets of the data and the attribution method‚Äôs predictions of those outputs.
