quanda.benchmarks.downstream_eval.class_detection module

class quanda.benchmarks.downstream_eval.class_detection.ClassDetection(*args, **kwargs)[source]

Bases: Benchmark

Benchmark for class detection task. This benchmark evaluates the effectiveness of an attribution method in detecting the class of a test sample from its highest attributed training point. Intuitively, a good attribution method should assign the highest attribution to the class of the test sample, as argued in Hanawa et al. (2021) and Kwon et al. (2024).

References

1) Hanawa, K., Yokoi, S., Hara, S., & Inui, K. (2021). Evaluation of similarity-based explanations. In International Conference on Learning Representations.

2) Kwon, Y., Wu, E., Wu, K., Zou, J., (2024). DataInf: Efficiently Estimating Data Influence in LoRA-tuned LLMs and Diffusion Models. The Twelfth International Conference on Learning Representations.
