quanda.metrics.heuristics.top_k_overlap module

class quanda.metrics.heuristics.top_k_overlap.TopKOverlapMetric(model: Module, train_dataset: Dataset, explainer_cls: type | None = None, expl_kwargs: dict | None = None, model_id: str | None = '0', cache_dir: str = './cache', top_k: int = 1, *args, **kwargs)[source]

Bases: Metric

Metric that measures the overlap of top-k explanations across all test samples, as argued in (1). A good data attributor’s attributions should depend on the input data, and thus should have low overlap between test points.

References

1) Barshan, Elnaz, Marc-Etienne Brunet, and Gintare Karolina Dziugaite. “Relatif: Identifying explanatory training samples via relative influence.” International Conference on Artificial Intelligence and Statistics. PMLR, 2020.
